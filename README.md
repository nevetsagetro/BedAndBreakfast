
# Bed & BreakFast

Responsive website made with Tailwind CSS, includes dark mode toggle.


## Features
- Responsive Layout
- Dark Mode toggle: to switch click on the moon icon.
    - located on the navbar (top on desktop version).
    - Mobile tab_bar (bottom on mobile version).
